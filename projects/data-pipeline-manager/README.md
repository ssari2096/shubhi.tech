# What is this project about?
Data Pipeline Manager is a cloud-native backend application for managing and monitoring data pipelines.

# Why I built this?
This project is being built as a hands-on project to learn and demonstrate modern cloud-native development using Python, FastAPI, Docker, Kubernetes, Google Cloud Platform.

# What is the end goal?
The goal is to simulate a production inspired service that allows users to create, manage, trigger and monitor data pipelines through REST APIs.

# TechStack
Python
FastAPI
SQL
Docker
Kubernetes
Google Kubernetes Engine(GKE)
GitHub Actions
Google Cloud Platform
BigQuery
Pub/Sub

# Features
Create data pipelines
View pipeline details
Update pipeline configuration
Delete pipelines
Trigger pipeline execution
Monitor execution status
View execution history
Health check API
Logging and monitoring
Authentication and authorization
Cloud Deployment on GKE
