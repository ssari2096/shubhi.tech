Adding my day to day learnings here and associated terms as a glossary:

1) Data Pipeline : A series of automated software steps that moves data from one place to another.
2) Cloud Native : Software approach for building, deploying, and managing modern applications designed specifically to take advantage of cloud computing environments.
3) FastAPI : FastAPI is a modern, fast (high-performance), web framework for building APIs with Python. Can be called as a tool which allows applications to talk to each other.
4) Docker : It is a tool that helps developers to bundle a software application and everything it needs to run into a single package called container.
5) Kubernetes : It is a system to manage which helps in automating the deployment, scaling and management of containerized applications.
6) RestAPIs : Allow two computers to exchange data securely over internet like GET, PUT, POST, PATCH and DELETE.
7) GKE : Google Cloud Kubernetes service, fully managed by Google.
